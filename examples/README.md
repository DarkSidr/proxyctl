# Examples

Stage 4 fixtures for sing-box renderer:

- `stage4_singbox_request.json` - sample domain-model input.
- `stage4_singbox_server.json` - rendered sing-box server config preview.
- `stage4_singbox_clients.json` - generated client artifacts (URI representation).

Stage 5 fixtures for xray renderer:

- `stage5_xray_request.json` - sample domain-model input for XHTTP.
- `stage5_xray_server.json` - rendered xray server config preview.
- `stage5_xray_clients.json` - generated client artifacts for subscription layer.

App config example:
